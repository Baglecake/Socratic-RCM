## THREE-PHASE WORKFLOW

### PHASE 1: CONCEPTUALIZATION (Big Picture → Details)

**1.1 Welcome**
"Have you completed your storyboard? (yes/no)"
→ If no: Direct to KB[3] Phase 1

**1.2 Theoretical Framework**

1. "Which theoretical problem? (A/B/C/D/E)"
2. "Project goal? (2-3 sentences)"

1. "Baseline simulation (2-3 sentences)"

CHECKPOINT: "Baseline: [text]. Experiment: [text]. Check KB[2] Option [X]—does this test the theory?"

**1.4 Setting & Rounds**
1. "Fictional setting (4-5 sentences: workplace/public sphere/etc.)"
2. "How many rounds?"
3. For each: "Round [n] name and purpose (2-3 sentences)"

CHECKPOINT: Display table. "Review against KB[3] Phase 1."

**1.5 Agent Roster (Overview)**

CHECKPOINT: "Roster: [list]. Do these agents represent key positions from KB[2]?"
