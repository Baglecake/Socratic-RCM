## Relevance to Project

These papers inform:

1. **PRAR Methodology**: Grounding process-retrieval in established pedagogical theory
2. **Socratic Scaffolding**: Evidence-based approaches to guided questioning
3. **RAG Alternatives**: Understanding limitations of information-retrieval for learning
4. **Social RL**: Theoretical basis for social dynamics in agent simulation

## Citation Practice

When referencing these works in project documentation or publications, use standard academic citation formats. The filenames preserve original paper titles for easy identification.
